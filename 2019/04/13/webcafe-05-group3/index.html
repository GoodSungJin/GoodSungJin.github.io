WebCafe Markup_05

class="group3" Markup



class="event-related" Markup

  • 편리한 CSS 작업을 위해 신규이벤트와 관련사이트를 하나의 <div>로 묶어주고, 각각 <section> 으로 지정해주었다.
  • <div class="event-detail"> 에 들어있는 내용을 읽은 후 동작이 필요하기 때문에 <div class="event-detail"> > <div class="event-btn"> 순으로 Markup 해주었다.
  • 나중에 CSS 작업을 위해 내용이 들어있는 <div class="event-detail">와 버튼이 들어있는 <div class="event-btn"> 으로 영역을 나눠주었다.
  • <button>의 경우 CSS를 이용해 sprites기법을 사용할 것이다.
    • sprites기법 : 이미지를 불러오는 방법중 하나로 서버로의 요청 횟수를 최소화할 수 있는 것이 장점이다.
